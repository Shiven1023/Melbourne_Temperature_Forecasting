# Melbourne Minimum Temperature Forecasting

## Project Overview
Comparing forecasting techniques on the Melbourne daily minimum 
temperature dataset (1981-1990).

## Models Compared
- Naive Forecasting
- Linear Regression
- ARIMA/SARIMA
- Simple RNN
- LSTM

## Project Structure
notebooks/   - Jupyter notebooks for each phase
src/         - Reusable utility functions
data/        - Raw and processed data
results/     - Saved figures and metrics
